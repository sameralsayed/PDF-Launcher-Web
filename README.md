# PDF & Launcher Web

A fast, lightweight, and private PDF Viewer & Launcher that runs entirely in your browser.

![PDF Launcher Preview](https://via.placeholder.com/800x400/4e54c8/ffffff?text=PDF+%26+Launcher)

## ✨ Features
- ⚡ **Instant PDF opening** — blazing fast with no lag
- 📁 **Clean PDF Library** — launcher-style list of your documents
- 🗑️ **Batch delete** — select and remove multiple PDFs
- 🔒 **100% Local & Private** — nothing is uploaded to any server
- 🌙 **Dark / Light mode** — easy on the eyes
- 📱 **Fully responsive** — perfect on mobile, tablet, and desktop
- 🔎 **Powerful viewer** — zoom, search, thumbnails via pdf.js

## 🚀 How to Run
1. Download or clone the repository
2. Open `index.html` in any modern browser
3. Upload your PDF files and start reading

No installation or server required.

## 🛠️ Built With
- HTML5
- CSS3
- Bootstrap 5
- jQuery
- pdf.js (Mozilla)
